 Student Management System (Python CLI)

A simple Command Line Interface (CLI) based Student Management System built using Python.  
This project allows users to manage student records efficiently.

Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save Data to File

Technologies Used

- Python
- File Handling
- JSON (optional upgrade)
- CLI (Command Line Interface)

Project Structure

student-management-system.v2/
│  
├── main.py  
├── students.json  
├── README.md  
└── .gitignore  

## How to Run

1. Clone repository

git clone https://github.com/shashankA137/student-manager-python-V2.git

2. Navigate to project

cd student-manager-python-V2

3. Run program

python main.py
